# 3D-Manual
3D Manual, CAD chatbot

This repository contains a web-based AI chatbot that answers questions based on 3D printer and Fusion 360 manuals.
It uses the OpenAI GPT API to generate responses grounded in uploaded documentation.
The chatbot supports multi-user access and conversational memory.
